# Prompts — Illustrations poissons manquants

## Style de référence (à appliquer sur TOUS les prompts ci-dessous)

Les illustrations existantes (voir `public/images/fishes/`) sont dans un style :
- Peinture naturaliste de type guide ornithologique / planche botanique
- Fond **transparent** (PNG)
- Éclairage latéral doux, reflets sur les écailles bien détaillés
- Rendu photoréaliste-peint : on voit les textures d'écailles, la translucidité des nageoires
- Format paysage, poisson orienté vers la gauche
- Résolution : 800×400 px minimum

**Colle ce bloc de style au début de chaque prompt :**

> Naturalistic fish illustration in the style of a vintage field guide painting, transparent background, PNG, soft lateral lighting, detailed scales and fins, translucent fin membranes, photorealistic painterly style, the fish facing left, clean white/transparent background, no shadow, scientific illustration quality
